# WhatsApp Checker

Простой статический одностраничный сайт для быстрой проверки номера в WhatsApp.

Файл: `index.html` — вся логика и стили находятся внутри этого файла.

Ключевые элементы:

- Поле ввода: `id="phone"` — ожидается номер с кодом страны (например, `+79001234567`).
- Сообщение об ошибке: `id="error"`.
- Функция: `openWhatsApp()` формирует ссылку `https://wa.me/{cleanPhone}` и открывает её в новой вкладке.

Как проверить локально

1. Открыть файл напрямую (двойной клик) или запустить локальный сервер.

   Python 3 (PowerShell):

   ```powershell
   python -m http.server 8000
   Start-Process 'http://localhost:8000'
   ```

   или (если установлен Node):

   ```powershell
   npx serve . -l 5000
   Start-Process 'http://localhost:5000'
   ```

2. Ввести номер и нажать кнопку «Открыть в WhatsApp» или нажать Enter.

Тест-кейсы

- Корректный номер: `+79001234567` → откроется `https://wa.me/79001234567`.
- Номер с форматированием: `+7 (900) 123-45-67` → очистится до `79001234567`.
- Слишком короткий: `+123` → появится сообщение об ошибке и красная рамка.

Примечания для разработчиков / AI-агентов

- Вся логика в `index.html` — правьте аккуратно и добавляйте комментарии на русском рядом с изменениями.
- Если меняете валидацию длины номера, обновите оба условия в `openWhatsApp()` (перед и после очистки номера).
